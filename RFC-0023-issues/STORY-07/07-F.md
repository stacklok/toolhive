### Migration Guide

Write a v1alpha1 → v1alpha2 field mapping table:

| v1alpha1 Field | v1alpha2 Field | Notes |
|---|---|---|
| `spec.port` | `spec.proxyPort` | Direct rename |
| `spec.targetPort` | `spec.mcpPort` | Direct rename |
| `spec.toolsFilter` | `spec.toolConfigRef` | Use MCPToolConfig CRD |
| `spec.oidcConfig.type` + `spec.oidcConfig.inline` | `spec.oidcConfig.inline` | Remove `type` discriminator |
| `spec.oidcConfig.inline.clientSecret` | `spec.oidcConfig.inline.clientSecretRef` | Move to Secret |
| `spec.oidcConfig.inline.thvCABundlePath` | `spec.oidcConfig.inline.caBundleRef` | Move to ConfigMap |
| (new) `spec.oidcConfig.ref` | Reference MCPOIDCConfig | Extract shared config |
| (new) `spec.telemetryConfig.ref` | Reference MCPTelemetryConfig | Extract shared config |
