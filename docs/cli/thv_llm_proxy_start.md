## thv llm proxy start

Start the LLM gateway localhost proxy

### Synopsis

Start a localhost reverse proxy that injects fresh OIDC tokens for AI tools
that only accept static API keys (e.g. Cursor).

The proxy runs in the foreground and blocks until interrupted (Ctrl+C).
To run it in the background, use your shell or a process manager:

  thv llm proxy start &

```
thv llm proxy start [flags]
```
