## thv start

Start a tooling server

### Synopsis

Start a stopped tooling server managed by ToolHive.
If the server is not running, it will be started.
Supports both container-based and remote MCP servers.

```
thv start [workload-name] [flags]
```

**Aliases:** `restart`

### Options

```
  -a, --all            Start all MCP servers
  -f, --foreground     Run the started workload in foreground mode
  -g, --group string   Start all MCP servers in a specific group
  -h, --help           help for start
```
