## thv plugin uninstall

Uninstall a plugin

### Synopsis

Remove a previously installed plugin by name.

```
thv plugin uninstall [plugin-name] [flags]
```

### Options

```
  -h, --help                  help for uninstall
      --project-root string   Project root path for project-scoped plugins
      --scope string          Scope to uninstall from (user, project) (default "user")
```
