# CRD Helm Wrapper

A Go tool that wraps Kubernetes CRD YAML files with Helm template conditionals for:
- **Conditional installation** (`crds.install`)
- **Skip functionality** (`crds.skip`)
- **Resource policy annotations** (`crds.keep` → `helm.sh/resource-policy: keep`)

## Why This Tool?

Raw CRDs generated by `controller-gen` don't have Helm templating. This tool wraps them with conditionals so users can:
- Skip CRD installation entirely
- Skip specific CRDs (useful for upgrades or external management)
- Prevent Helm from deleting CRDs on uninstall

## Usage

```bash
# Build the tool
go build -o crd-helm-wrapper ./tools/crd-helm-wrapper/

# Run it
./crd-helm-wrapper \
  -source deploy/charts/operator-crds/files/crds \
  -target deploy/charts/operator-crds/templates \
  -verbose
```

## Flags

| Flag | Description | Required |
|------|-------------|----------|
| `-source` | Source directory containing raw CRD YAML files | Yes |
| `-target` | Target directory for wrapped Helm templates | Yes |
| `-verbose` | Enable verbose output | No |

## Output Format

Given a CRD like:

```yaml
apiVersion: apiextensions.k8s.io/v1
kind: CustomResourceDefinition
metadata:
  annotations:
    controller-gen.kubebuilder.io/version: v0.17.3
  name: mcpservers.toolhive.stacklok.dev
spec:
  ...
```

The tool produces:

```yaml
{{- if .Values.crds.install }}
{{- if not (has "mcpservers.toolhive.stacklok.dev" .Values.crds.skip) }}
# Source: {{ $.Chart.Name }}/templates/toolhive.stacklok.dev_mcpservers.yaml
apiVersion: apiextensions.k8s.io/v1
kind: CustomResourceDefinition
metadata:
  annotations:
{{- if .Values.crds.keep }}
    helm.sh/resource-policy: keep
{{- end }}
    controller-gen.kubebuilder.io/version: v0.17.3
  name: mcpservers.toolhive.stacklok.dev
spec:
  ...
{{- end }}
{{- end }}
```

## Values Configuration

The wrapped CRDs expect these values:

```yaml
crds:
  # Whether to install CRDs as part of the Helm release
  install: true
  # Whether to add the "helm.sh/resource-policy: keep" annotation
  keep: true
  # List of CRD names to skip during installation
  skip: []
```

## Integration with Task

This tool is meant to be run as part of `task operator-manifests`:

```yaml
# In Taskfile.yml
operator-manifests:
  cmds:
    - controller-gen crd paths="./..." output:crd:artifacts:config=../../deploy/charts/operator-crds/files/crds
    - go run ./tools/crd-helm-wrapper -source deploy/charts/operator-crds/files/crds -target deploy/charts/operator-crds/templates
```
